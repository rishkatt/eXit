# eXit

![img0](images/eXit.png)

## Requirements

### Software

- Python 3.8 or greater

### Packages

- os
- sys
- time

## Usage

If you are operating in Linux:
```
python3 exit.py
```

If you are operating in Windows:
```
python exit.py
```

At any point during the game, if you would like to quit, use:
- q
- q()
- quit
- exit
